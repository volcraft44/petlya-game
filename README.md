# 🌀 ПЕТЛЯ — Dungeon Roguelike

Пиксельный рогалик-платформер на Godot 4.4 с dream-pop визуалом, петлёй времени,
CS/ULTRAKILL-механиками и психоделической атмосферой.

---

## ⬇️ СКАЧАТЬ И ИГРАТЬ

### [🎮 СКАЧАТЬ PETLYA.exe](https://github.com/vulcraft2012-beep/dungeon-roguelike-godot/releases/download/v1.0/PETLYA.exe)

Скачай файл, **запусти двойным кликом** — и играй. Godot НЕ нужен,
ничего устанавливать не надо. Вся игра внутри одного .exe.

> При первом запуске Windows может спросить разрешение —
> нажми «Подробнее» → «Выполнить в любом случае».

### Для разработчиков — открыть исходники
1. [Скачать ZIP с кодом](https://github.com/vulcraft2012-beep/dungeon-roguelike-godot/archive/refs/heads/master.zip)

## 🎮 Управление

| Клавиша | Действие |
|---|---|
| **WASD** | Движение |
| **Space** | Прыжок / Perfect Bhop |
| **Shift + Space** | Long Jump (низкий, дальний) |
| **C** | Кувырок |
| **LCtrl** | Dash (3 заряда) |
| **ЛКМ** | Атака |
| **ПКМ** | Щит / Scope (с AWP) |
| **G** | Дымовая граната |
| **V** | Флешка |
| **Q** | Inspect оружия (после простоя) |
| **E** | Взаимодействие |
| **H** | Лечение |

## ✨ Фичи

- **CS-механики**: рарность оружия, хедшоты, AWP со scope, дымовухи, флешки
- **ULTRAKILL стиль-ранги** (D → U) с боевыми бонусами
- **Bhop** с perfect-таймингом + dash-чарджи
- **Кастомная игра** — выбор карт, свитков, оружия, событий
- **Dream-pop визуал** — пастель, voxel-тайлы, bloom, параллакс
- **Элитные враги** с аффиксами
- **Петля времени** — сюжет, лор на стенах
